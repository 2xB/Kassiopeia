External references#

To cite the use of and/or data produced with the assistance of Kassiopeia or its associated libraries (KGeoBag, KEMField, or Kommon). Please refer to the paper:

D. Furse et al. (2017) New J. Phys. 19 053012: “Kassiopeia: A Modern, Extensible C++ Particle Tracking Package” (doi:10.1088/1367-2630/aa6950)

In addition to this user guide, Kassiopeia and its associated libraries have been documented extensively in several PhD theses. Many of these can be found under the list of KATRIN publications.

Developer contact#

To make bug reports, propose additional features and/or patches, please use the pull-request and issue tracker available on the Kassiopeia GitHub repository here. Additional unrelated inquiries may be directed towards: katrin-kasper (at) lists (dot) kit (dot) edu.
