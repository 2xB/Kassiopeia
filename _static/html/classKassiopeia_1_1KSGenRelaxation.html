Detailed Description

Applicable to singly ioized atoms, ie. with a vacancy in the K to x(depends on input file) shell. The class relaxes the atom by creating auger electrons and fluorecene photons until there is no vacancy left. The code is based on inputfiles by penelope (see docu page 230.)
