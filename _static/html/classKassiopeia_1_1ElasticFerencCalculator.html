Detailed Description

class to calculate elastic electron - hydrogen Scattering

Author
F. Glueck. Put into this class by S.Mertens

Detailed Description: This class contains methods to calculate:

  • elastric cross section in m^2
  • energy loss (in eV) and scattering angle (in rad) for elastic scattering
  • differential cross section for elastic scattering in m^2/steradian

differential cross section for elastic scattering in m^2/steradian

This subroutine computes the differential cross section Del= d sigma/d Omega of elastic electron scattering on molecular hydrogen in m^2/steradian. See: Nishimura et al., J. Phys. Soc. Jpn. 54 (1985) 1757.

Parameters

cosThetacos(Theta), where Theta is the polar scatt. angle
